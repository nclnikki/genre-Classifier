Music Genre Classification

Welcome to Music Genre Classifier!
This application uses machine learning to classify music genres based on audio files. Upload an audio file, and the system will predict its genre.


Note: Due to the limitations of Vercel, larger file size might not be compatible.
Enjoy!


About the Project

This project uses Convolutional Neural Network (CNN) architecture to analyze audio data.
Dataset used to train the Music Genre Classification model for this project is from GTZAN dataset
